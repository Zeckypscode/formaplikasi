Aplikasi ini merupakan proyek Ujian Tengah Semester (UTS) untuk mata kuliah Pemrograman Mobile.
Aplikasi ini berfungsi sebagai form pendaftaran seminar yang mencakup fitur login, input data pendaftar (nama, gender, hobi, email, dll), serta menampilkan ringkasan data hasil pendaftaran.

## Fitur Utama
Halaman Login: Autentikasi sederhana untuk masuk ke aplikasi.
Validasi Form Real-time: Memastikan input seperti Email, Nomor HP, dan Password sesuai format sebelum dikirim.
Pilihan Seminar: Menggunakan Spinner untuk memilih topik seminar yang tersedia.
Dialog Konfirmasi: Memastikan user yakin dengan data yang diinput sebelum berpindah halaman.
Halaman Hasil: Menampilkan data yang telah diinput menggunakan Intent.

## Teknologi yang digunakan
IDE: Android Studio.
Bahasa Pemrograman: Java.
Android API: API 33 (Android 13.0 Tiramisu).
Version Control: Git & GitHub.

## Struktur Proyek
MainActivity.java: Menangani logika form dan validasi input.
ResultActivity.java: Menampilkan ringkasan data pendaftaran.
LoginActivity.java: Menangani tampilan awal sebelum masuk ke form.
strings.xml: Menyimpan semua resource teks aplikasi secara terpusat.

## Cara Menjalankan
Clone repository ini.
Buka project menggunakan Android Studio.
Pastikan sudah menginstal SDK Android API 33.
Run menggunakan Emulator (AVD) atau device fisik.
